# Personal Finance Tracker

## Project Description
A modular Python-based personal finance tracking system that allows users to:
- Add and categorize expenses
- Store data using JSON
- Generate reports
- Track spending patterns
- Backup and restore data

## Features
- Expense validation
- File persistence with backup system
- Category breakdown reports
- Monthly tracking
- Modular architecture
- Basic unit tests
